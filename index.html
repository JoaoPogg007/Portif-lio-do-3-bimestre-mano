Meu Jogo: Descubra quem é o assassino!

O projeto é uma página web criada com HTML e CSS, destinada a exibir um jogo para descobrir um assassino entre algumas pessoas na mansão.

O HTML organiza o conteúdo em um cabeçalho com o título do site, com uma edição mais elaborada das perguntas e imagens que o site pode ter.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos.

O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração do cenário.

Meu Jogo: Cards do conhecimento

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simula uma experiência divertida e que incentiva seu conhecimento sobre coisas gerais, desde línguas estrangeiras, até perguntas de programação.

O HTML estrutura o conteúdo da página, dividindo-o em seções como os cards, e as formas dos cards, deixando a estrutura melhor elaborada para os cards.

O CSS ajusta as cores dos cards, do fundo, da página inteira, deixando uma estética mais personalizada e única em um jogo.

Os scripts JavaScript incluem módulos para gerar e animar os cards.

O App.js ajuda na escolha das perguntas, apoiando na escolha delas, editando manualmente ou de forma aleatória.
